#### PS

微信中有一些API是要配合xml的标签使用，例如音频，视频，动画。所以，动画类的API并没有在本插件的自动补全范围，可以查阅官方文档进行编写。

[微信开放平台](https://mp.weixin.qq.com/debug/wxadoc/dev/api/canvas/intro.html)
